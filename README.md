## 📌 Descripción

SmartTask AI es una aplicación desarrollada para ayudar a estudiantes y profesionales a administrar sus tareas diarias de manera eficiente.

El sistema permite:

- ✅ Crear y organizar tareas
- 📅 Gestionar horarios
- 🤖 Obtener recomendaciones inteligentes
- 📊 Visualizar progreso y productividad
- 🔔 Recibir recordatorios automáticos
